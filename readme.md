## option:
right mouse button:undo,redo,cut,copy,past,delete,save to file,load data from file

## keyboard shortcuts:
ctrl+z => undo,
ctrl+y => redo,
ctrl+x => cut,
ctrl+v => past,
del    => delete,
ctrl+s => save to file,
ctrl+l => load data from file
